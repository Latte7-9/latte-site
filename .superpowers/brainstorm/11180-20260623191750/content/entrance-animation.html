🎥 电影感揭幕

标题从模糊到清晰 + 光晕扫过 + 粒子汇聚成文字。像电影片头般有仪式感。

GSAP实现：filter:blur → 0, 光条 sweep, 粒子 stagger 汇聚
时长：约 2-3 秒，自动或点击跳过

Latte

💧 液态形变

不规则有机形状（blob）从色块慢慢形变成文字或Logo，带流动感。

GSAP实现：SVG/CSS border-radius 形变 + clip-path 动画
时长：约 3-4 秒

🧩 像素碎片重组

画面被切割成像素块散开 → 旋转飞入重组成完整标题。赛博感爆炸。

GSAP实现：Canvas/CSS Grid 像素块 stagger + 旋转 + 位移
时长：约 1.5-2 秒，节奏快、冲击力强
