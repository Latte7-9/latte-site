🎨 三种液态变体方案

Latte

🫧 墨滴扩散

一滴发光液体从中心滴落 → 扩散成不规则形状 → 文字从液体中浮现。
像牛奶滴入咖啡的慢镜头，呼应"Latte"意象。

背景：纯黑 + 微弱的径向光晕
色彩：蓝(#2990c0) → 紫(#7c4dff) 渐变光
音效提示：水滴声 + 低频嗡鸣

Latte

🌌 极光流彩

多层彩色光幕像极光般在屏幕上游动飘荡，逐渐凝聚收拢成标题。
梦幻、空灵、高级感。

背景：深空黑渐变
色彩：蓝 → 紫 → 青 多色渐变叠加
音效提示：空灵的合成器pad音

Latte

🎗️ 黏稠撕拉

画面中间出现一条发光的"液线"，上下拉开 → 像撕开一层膜 → 露出背后的标题内容。
质感像蜂蜜拉丝，有粘稠的重量感。

背景：纯黑 → 拉开的裂缝中透出光
色彩：冷暖光双边对峙（蓝上紫下）
音效提示：低频撕裂声 + 嗡鸣

🛠️ GSAP 实现要点
关键API
  • gsap.to() — border-radius 连续形变
  • gsap.timeline() — 分阶段编排
  • MorphSVGPlugin — SVG路径变形（可选）
  • CustomEase — 模拟液体粘滞缓动
性能优化
  • 仅动画 border-radius + opacity + scale
  • 使用 CSS will-change: border-radius, filter
  • 粒子数量控制在 20-30 个
  • 入场结束后 kill 所有 tween
