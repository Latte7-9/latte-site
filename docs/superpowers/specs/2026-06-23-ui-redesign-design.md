# Latte 个人网站 UI 大改造 — 设计文档

> **日期**：2026-06-23  
> **版本**：3.0  
> **设计基调**：毛玻璃赛博 × 霓虹暗夜

---

## 一、设计目标

在**保留全部现有功能和板块**的前提下，全面重塑视觉风格：

- 从暗色科技风 → 毛玻璃赛博 × 霓虹暗夜
- Anime.js → GSAP 3.x 动效引擎
- 保留：11 个板块 + 黑胶唱片播放器 + 橡皮鸭留言水池
- 新增：墨滴扩散入场动画 + 8 项点睛增强

## 三、入场动画：墨滴扩散

### 3.1 分镜时序 (总计约 3.2s)

| 阶段 | 时间 | 描述 |
|------|------|------|
| 暗场 | 0.0s | 纯黑全屏，中心微弱液态光斑呼吸 |
| 膨胀 | 0.5s | 光斑缓缓变大，边缘不规则摆动 |
| 形变 | 1.5s | 光斑剧烈扭曲，border-radius 持续变化，蓝→紫渐变 |
| 凝结 | 2.5s | 液团散开，文字从内部浮现，粒子飘散 |
| 沉降 | 3.2s | 副标题 + 向下箭头出现，等待用户交互 |

### 3.2 GSAP 实现要点

- 使用 gsap.timeline() 分阶段编排
- 核心属性：border-radius、scale、opacity、filter:blur()
- 粒子：20-30 个霓虹色小点，stagger 飘散
- 入场结束后 kill 所有 tween

## 四、页面结构 & 板块排列

### 4.1 浏览流线 (由上至下)

`
导航栏 (悬浮毛玻璃)
  ↓
关于我      — 左头像 + 右文字
  ↓
博客预览    — 横向滚动卡片 x3
  ↓
兴趣 | 正在翻阅  — 双栏并排
  ↓
随机听听    — 居中大卡片
  ↓
留言板      — 水池 + 霓虹鸭群 + 表单
  ↓
联系方式    — 图标横排 + 邮箱
  ↓
页脚
`

### 4.2 各板块排版详规

**导航栏** — 悬浮毛玻璃卡片，居中显示。左: 品牌名 Latte, 右: 首页/博客链接。固定顶部，滚动微缩，当前页霓虹高亮。

**关于我** — 左侧圆形头像 (霓虹光环边框), 右侧文字左对齐。整体居中，无外框卡片直接融入背景。

**博客预览** — 横向滚动卡片列表，每卡片含标题+日期+摘要。渐变细边框，hover 时边框霓虹发光。右上角 "查看全部 →" 链接。

**兴趣 + 正在翻阅 (双栏)** — 左栏: 兴趣标签云 (霓虹色药丸标签)。右栏: 书籍小封面+书名+作者纵向排列。各自独立渐变边框。移动端自动变为上下单栏。

**随机听听** — 居中单张大卡片 (约 340px 宽)。专辑封面 → 歌名 → 艺术家 → 渐变播放按钮。

**留言板** — 容器底部渐变水域底色。霓虹色橡皮鸭漂浮。底部简洁表单: 名字+内容+"丢鸭子"按钮。点击鸭子弹出毛玻璃气泡卡片。

**联系方式** — 图标居中横排，毛玻璃方块底。hover 霓虹发光。邮箱文字链接。

**页脚** — 顶部细线分隔。"© 2026 Latte · 用 Codex 搭建"

## 七、技术实施

### 7.1 动效引擎
- GSAP 3.x (替代 Anime.js)
- ScrollTrigger 插件 (滚动触发)
- CustomEase (液体缓动入场)
- gsap.matchMedia (响应式断点)

### 7.2 CSS 技术
- backdrop-filter: blur() 毛玻璃
- mask-composite 渐变边框
- CSS 自定义属性 (--neon-pink, --neon-cyan 等)
- 保留 WebGL 粒子背景 (升级颜色)

### 7.3 架构决策
- 保留原生 HTML/CSS/JS 架构 (不迁移框架)
- CSS 按板块模块化组织
- 响应式: 移动端 < 768px 自适应

### 7.4 实施前必须执行
- 先存档当前版本 (踩坑日志 #005 教训)
- git pull 拉取最新代码 (踩坑日志 #008 教训)
- 在新目录或新分支开发
